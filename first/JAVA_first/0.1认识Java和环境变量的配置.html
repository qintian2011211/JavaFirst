2.程序语法分析：
·程序作用是描述生活中的物体的！

·生活中物体是分类的，每类物体有自己的名词和形容词的介绍学名叫“属性”，还有自己的动作
"物体是由属性+行为构成！"

·描述小狗！

语法：
public class 类名{
  代码...    
}
"框架!"
物体的动作对应程序中的"方法"
方法很多，有一个开始的方法，叫main方法！
语法:

public class Dog{
public static void main(String[] args){
  程序开始...
  打印一句话
  System.out.println("你好我的...");
}
}
·public【pa be lei ke】公共的。控制权限
·class 类的关键字！
·Dog  类名，一般首字母都大写！必须和文件名一致！
·{} 表示开始和结束
·控制权限：
  public 公共的--控制类的权限！
  static 静态的，控制功能的权限
·void 空，表示没有返回值/结果！
  【任何功能/动作，都会有结果的，无结果表示为void】
·main 是程序主入口名字！
·()中是功能的参数.
   main方法，参数是固定的是"一组字符串"
   String[] 一组字符串！
   a b c  开 关\你 -->单个字符
   abc开关你----->一串字符，String
·系统打印命令：
  System--系统
  out ----输出
  print----打印
  println()... 打印完后换行！
  ("打印内容") 
·[] 表示一组

方法的三要素
(1)访问修饰符 public 
(2)返回值 无返回值用void表示
(3)参数列表 ()中写参数

特殊符号：
\n: 换行
\t: 制表符，泛指8个空格！
\n,\t必须写在""里面！


·为程序添加说明文字"注释"
3种注释
(1)单行注释 //说明内容
(2)多行注释 /*内容*/  ---》一般用在类和方法上！
   /*
      内容1
      内容2..
    */ 
(3)文档注释--->用于类和方法  【使用】
   /**
    @作者 zzy
    @版本 1.0
    */

常见错误：
(1)类名和文件名一致,一般首字母大写
(2)返回值必须加。无 void表示
(3)java大小写敏感
(4)字符串用""引起来
(5)命令结束用; 结束
【所有符号必须是英文的！】
