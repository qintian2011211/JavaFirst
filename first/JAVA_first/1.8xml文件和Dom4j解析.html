1.定义xml文件:
  (1)xml：可扩展标记语言，用于存储数据
     优点：结构清晰,体积小
  (2)定义要求：   
     <1>要有声明语句,而且要在第一行:
       
     <2>一个xml文件只能有一个根元素(根标签)
     <3>标签要正确嵌套,且标签名区分大小写
     <4>属性值要用双引号引起来
     <5>要存的值可以以属性值或者标签中的值,
        两种方式进行保存。
  语法:
     <标签名 属性名="属性值">
        <标签名>值
             
  示例：
    
      java开发

dom解析：
  实现步骤：
    1.创建解析器工厂
    DocumentBuilderFactory factory=
      DocumentBuilderFactory.newInstance(); 
    2.获得解析器对象
      DocumentBuilder builder=
          factory.newDocumentBuilder();
    3.获得Document对象,表示整个xml
      org.w3c.dom.Document doc=
          builder.parse("xml文件路径");    
    4.获得根节点：
     Element root= doc.getDocumentElement(); 
    5.解析xml
      常用方法：
         getNodeName():获得节点名
         getNodeType():获得节点类型
         item(int index):遍历节点集合
         getTextContent:获得最后一层节点中的内容
         getLength():获得集合长度
 小结：dom解析中使用的类：
   Document,NodeList,Node,Element
   6.获取标签中的属性值：
      //将node节点对象强转为Element
     示例：
        Element e=(Element)node;//node是Node类的对象
        e.getAttribute("属性名");//获得标签中指定的属性值
