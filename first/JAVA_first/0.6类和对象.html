2.方法
物体的功能---》程序方法！
小狗做运算！
·求固定数和（1+1）
·求任意两个数和
·在任意两个数和的基础上再运算！

语法：
(1)无参无返回值方法
访问修饰符 void 方法名(){....}
public void jiSuan(){...}
(2)有参，无返回值
访问修饰符  void 方法名(参数1，参数2,..){...}
public void jiSuan2(int num1,int num2){...}
(3)有参有返回值
访问修饰符 返回类型 方法名(参数1,..){
  return 返回值；
}
(4)无参有返回值
public  int jiSuan3(){ return 1+1;}
【注意：参数只表示类型，不代表具体值---》形参】
【注意2：带返回值方法，可以当成结果被使用！】
