2.正则
·定义：是一种验证字符串规则的表达式！
·作用：做字符串校验！
·应用：各种网络输入中...
·正则表达式可用于任何一门高级语言：
  比如：java,objectC,php,.net,网页js,
  ...
·使用：
  字符串.matches(正则)-->成功true
                         失败：false
1.正则表达式：
(1)是一种字符串验证规则，专门用于做字符串的格式校验
   比如：校验电话号码1[3,4,5,7,8]******         验证邮箱。。。等
(2)英文：
   Regular  Expression 正则表达式，简写regx
   【rui gei ou le】  【ei ke si pu ruai sen】
·规则：
\\ ----- \
^  ----- 开始在[]中表示非
$  ------结束
 开始和结束一般省略
\d ----- 0-9的数字等同[0-9]
[] ----- 表示范围
[^]----- 不在该范围 [^0-9]
\w ----- 字母数字下划线
\W ----- 非字母数字下划线
\s ----- 1个空白
\S ----- 1个非空白
...更多参考PPT

验证手机号规则："1[3,4,5,7,8]\\d{9}"
