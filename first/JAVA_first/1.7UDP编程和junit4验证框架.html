1.基于UDP协议编程
  特点：只负责发送数据，不负责数据是否被接收
        安全性较低
  实现方式：
  (1)发送方
    //1.创建DatagramSocket,不需要指定端口号
    DatagramSocket socket=new DatagramSocket();
    //2.创建数据包：DatagramPacket
    InetAddress ia=InetAddress.getByName("接收方的ip地址");
    DatagramPacket dp=
      new DatagramPacket(字节数组,长度,ia,端口号);
    //3.发送数据
      socket.send(dp);
  (2)接收方
    //1.创建DatagramSocket,要指定端口号
    DatagramSocket socket=new DatagramSocket(端口号);
    //2.创建DatagramPacket
    DatagramPacket dp=new DatagramPacket(字节数组,数组长度);
    //3.接收数据
    socket.receive(dp);
    //4.将字节数组转换成字符串
    String s=new String(dp.getData(),0,dp.getLength());
    
补充：当建立好一次连接之后,返回响应信息时不需要指定端口号
   SocketAddress address=dp.getSocketAddress();
   dp=new DatagramPacket(字节数组,字节长度,address);    
  说明：dp是DatagramPacket的对象。
