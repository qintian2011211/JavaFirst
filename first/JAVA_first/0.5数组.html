1.数组存储数据
·因为变量存储只能存储1个数据，有局限！
·Array:【ai rui】数组
·定义：数组是内存中连续的存储空间！
      (一组连续的变量！)
·特点
(1)数组可以存储任意类型
(2)数组中元素位置从0开始
   索引/下标
(3)数组长度：数组名.length属性！
(4)根据下标使用数组
   arr[0]
   如果索引值过大，会报数组下标越界！
   ArrayOutOfBoundsException
·语法
(1)  数据类型[] 名字= {值1，值2....};
  int[] ages ={20,22,23,19,...}
(2)  数据类型[] 名字;
  名字[0]=值1;
  名字[1]=值2;
  ...
(3)  数据类型[] 名字= new 数据类型[大小];
     int[] ages = new int[4];

2.数组默认值：
·int[] a = new int[4];
  new后(创建)数组后，会自动开辟可以存储4个值的空间！默认每个都保存int类型！默认值为0
·int[] a;  【没new,没有开辟内存空间！】
  a[0]-->报错！空指针异常！
               nullPointerException
·int[] a ={};
  a[0]--->数组下标越界
           
3.for循环简写
·foreach循环
  for(元素类型 别名:数组名){
    sysout....(别名);
  }
  别名就相当于数组中的每个值！
  应用：循环编列所有
4.数据工具类Arrays
方法
·排序：Arrays.sort(数组) 升序
·查找：Arrays.binarySearch(数组,值)
  返回位置下标！
5.基本数据类型和引用数据类型区别
·数据类型分2类
  基本数据类型，引用数据类型
  【注意：String是引用数据类型，因为比较常用，String a = "aa"存储方式和基本数据类型一致！】
·基本数据类型直接存储在栈中！
·引用数据类型/对象类型，分2个地方存储
  地址/名字： 堆中！
  值：        栈中！
  String a =new String("aa");a分2个地方保存！
