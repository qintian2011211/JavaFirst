集合：
  List集合:
     存储特点：不唯一，有序
     arrayList:顺序存储,遍历或随机访问时效率高
     LinkedList:链式存储,插入或删除数据效率高
 共有方法：
    添加：add(Object obj)
    删除：remove(int index)
    获得数据：get(int index)
    更新数据：set(int index,Object obj)
    是否为空：isEmpty()
 LinkedList特有方法：
    addFirst(Object obj)
    addLast(Object obj)
    getFirst()
    getLast()
    removeFirst();
    removeLast()
 注：调用LinkedList特有方法时,声明的是LinkedList的对象   
  set集合:
     存储特点：唯一，无序

  1-14:
   (1)  Map集合：
        存储特点：使用key-value(键-值对)存储方式
        访问数据时通过key值查找value值
        key和value都是Object类型，但是实际开发中
        通常会使用String作为key,对象作为value 
     注：key不允许重复，value值可以重复
     常用方法：
       1.存值：put(Object key,Object value)   
       2.取值：get(Object key)
       3.获取key值集合：keySet();
       4.获得value值的集合：values()
       注：返回value值的集合是Collection类型
           然后调用toArray()方法,返回对象数组
       Object[] obj=map.values().toArray();
       5.判断key值是否存在：
           containsKey(Object key)
       6.判断value值是否存在：
           containsValue(Object value)
       7.判断是否为空：
          isEmpty();
       8.获得集合大小
          size()
       9.清空集合
          clear()
       10.通过key值删除对象
          remove(Object key)
   练习：创建一个教师类，包含教师的名字和所授课程

         然后遍历三名教师的信息。
         提供用户一个可以查询教师是否存在的方法           
   (2)泛型：
     作用：使存放到集合中的数据只能按照规定类型存储
           保证了集合中数据类型的统一 
     本质：参数化类型
     语法：<数据类型>
     示例：
       //该集合中只能存储Student对象
       List list=new ArrayList(); 
     
     泛型的种类：
       <1>泛型接口
          public interface 接口名<泛型参数>{
           
          }
       示例：
          public interface inter{
            public T  test(T t);
          }
          说明：T的位置表示使用的是同一种类型
       <2>泛型类：
          public class 类名{}
        示例：
          public class test{
            public void test(T t){}
          }
        注：泛型类多用于父类定义
       <3>泛型类的继承：
         1.面向对象的特性同样适用于泛型类，
           所以泛型类也可以被继承
	       2.继承了泛型类的子类，必须也是泛型类
	       示例：
	         public class Son extends Father{
	         }
