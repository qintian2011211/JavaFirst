·保存过程：先根据数据类型开辟一个存储空间,然后再保存数据！

6. 变量名规范
·变量名由字母数字下划线或$组成，但是首字母一定不能为数字！一般遵循"驼峰命名法"
 驼峰:第一个字母小写，每个新单词首字母大写
  zhangSanName
 一般使用英文全称，而且要有含义(语义)！
·变量不能是java中的关键字！
   int class =a;  错误！
   goto和const 是java预留的关键字，也不能用！！！
·变量不能重名！

 ++ 在前，先加1，再使用x

(2) ++在后，是比等号优先级别还低的运算符！

    y=x++; ==》先把x赋值给y,x再自加！
    y结果为3，x结果为4
