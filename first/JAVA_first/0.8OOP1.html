 私有化属性，对外提供公有的访问方法！

 共有：public

  alt+shift+s -->  r
·封装概念：
  私有化成员变量对外提供共有的get和set操作方法，
·目的：保证数据的安全性！

3. 简化逐一赋值？
·构造方法：
  方法名和类名相同，而且无返回值,这种特殊格式方法叫"构造方法"
·作用：赋值(简化赋值)
·语法：
 public 类名(){...}-》无参构造
 public 类名(参数列表。。){}--》带参构造
·默认每个类都有无参构造！
·自动生成快捷键
  alt+/--->无参构造
  alt+shift+s --- o  生成带参构造！

4.发现方法可以重名？？？
·java支持方法重载：
·重载：
  方法名相同，参数列表不同和返回值无关！
  参数列表不同：类型不同 或 个数不同
·前提：在1个类中！

  “一个类中方法重名的现象叫方法重载”
