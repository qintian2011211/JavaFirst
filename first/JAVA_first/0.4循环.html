3.语法1
循环： while
关键点分析：
·起始值
·判断条件(结束值)
·计数器改变起始的值

定义起始值
while(条件){
  .....
  改变起始值（迭代）
}

判断是否是偶数：　ｉ%2==0

·java调试！
(1)添加断点！
(2)单步运行F6
断点：程序中断运行的位置！

快捷键：单步运行F6
        进入方法：F5
        直接运行全部结束：F8
【注意观察3个窗口：变量变化窗口，控制台打印！断点窗口！】

·循环语法2：
 生活中案例：先做N遍练习，再测试！测试通过不用继续！
特点： 至少做了1次，while不能满足要求了

语法：
do{
  .....
  初始值改变 (变量迭代)
}while(条件)
案例：做100遍练习！
案例2：去做练习，检测如果会了，直接不再做！

·循环语法3：
  如果知道循环开始和结束，可以直接for循环！该循环结构非常清晰！
语法：
for(初始值;循环条件;变量迭代){
   。。。。。循环体！
}
案例 ：倒着打印1--100!
·循环中断！
 生活中需要：循环过程中需要中断/暂停，或强行终止！
程序实现关键字：
   中断/暂停：continue【ken tei niu】
    终止：break;
   在3种语法中都适用！
案例：循环跑圈！第3圈跑不动休息！
                第3圈因为身体原因强行中断！

 continue，中断当前循环。
 break:  终止整个循环！
