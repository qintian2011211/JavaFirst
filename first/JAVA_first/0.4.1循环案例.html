·数据存储扩展：
  变量只能存储1个值！
  如何存储多个值？比如40人的分数？
1 概念：内存中连续的存储空间叫“数组”
   数组:Array
2 数组定义语法：
·先定义后赋值
  int[] arr1;
  arr1={1,2,3};
·定义并指定大小，再赋值
  int[] arr1;
  arr1 = new int[2];
  arr1[0]=1;
  arr1[1]=2;  
·定义同时赋值
  int[] arr1 ={1,2,3};
3.特点:
(1)数组可以存储任意类型数据
   String[],int[],double[]...
   Object[] a={1,"aa",3.14}
(2)数组元素位置从0开始 
   元素位置学名叫"下标"
(3)根据索引来使用数组
   数组名[索引] arr1[0]
(4)数组长度 数组名.length属性
 数组名--》学名"标识符"
 元素位置-->索引
